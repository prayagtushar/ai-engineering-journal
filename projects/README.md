# Projects (small builds)

Self-contained learning builds — the full code lives **here in the journal**. Each one exists to learn a specific tool or pattern end-to-end. Larger, deployable builds live in their own repos and are showcased in [`../flagship-projects/`](../flagship-projects/).

| Project | What I built | Focus |
|---------|-------------|-------|
| [issue-tracker-fastapi/](issue-tracker-fastapi/) | A small issue-tracker REST API | FastAPI routing, Pydantic schemas, storage/middleware layering |
| [web-scraping/](web-scraping/) | A guided scraping pipeline (requests → pagination → API → Playwright → dataset) with [docs/](web-scraping/docs/) | HTML parsing, network interception, building a dataset |

## Convention

A build graduates to [`flagship-projects/`](../flagship-projects/) (its own repo) once it's multi-module, deployable, or worth presenting in an interview. Until then it lives here.
